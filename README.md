## Description
This game lets users catch flowers using arrow keys and earn points. It was inspired by fishing. My intention with this game design was to look relaxing but can actually get exciting with the time limit.

---
## Interaction Instructions
Describe how to use or play your project.
- The user sees a relaxing green/yellow gradient splash screen that is meant to look serene. They can start playing by clicking the spacebar or the "s" key.
- They can move the basket to start catching flowers using the left and right arrow keys.
